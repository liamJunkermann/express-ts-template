# Express-ts

## Build from source

1. Clone the repo

   ```sh
   git clone git@github.com:liamJunkermann/express-ts.git
   cd express-ts
   ```

2. Install dependencies.

   ```sh
   npm install
   ```

3. Build the production server.

   ```sh
   npm build
   ```

4. Run the server.
   ```sh
   npm start
   ```
